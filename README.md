## Sending message using itchat. 
### 功能简介
很简单，就是从特定网站上获取一些内容，例如天气预报/明言警句等通过微信发送给特定的人，比如父母/朋友。  

### 框架说明
在MessageHunter目录下实现了所有信息的获取:
- 获取天气预报的模块  
- 获取hitokoto网站一言的模块  
- 获取金山每日一句的模块   
- 工厂类模块, 负责对以上各种获取信息的模块进行了封装，只把工厂类的接口提供给外部用即可。
